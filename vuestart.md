* 初始化项目
```
# 全局安装 vue-cli
$ npm install --global vue-cli
# 创建一个基于 webpack 模板的新项目
$ vue init webpack my-project
# 安装依赖
$ cd my-project
$ npm install
$ npm run dev
npm install --save vuex
npm install --save sass-loader node-sass autoprefixer
```
